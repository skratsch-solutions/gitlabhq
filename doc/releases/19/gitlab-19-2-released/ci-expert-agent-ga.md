The CI Expert Agent is now generally available. This AI-powered assistant helps you create, debug, and optimize GitLab CI/CD pipelines. It analyzes your repository and provides guidance through your pipeline configuration.

The generally available (GA) release builds on the Beta and design-partner program, and brings the agent to production use with real customer workloads to reduce the barrier to CI/CD adoption in your projects.

The agent integrates with [Orbit](https://docs.gitlab.com/orbit/): when Orbit and its Knowledge Graph are enabled, the agent draws on graph-based code intelligence for more contextual pipeline recommendations. Orbit and the Knowledge Graph are in Beta.
