For GitLab Duo Agentic Chat, you can now configure an allowlist of approved models, and set an organization-wide default, if you are:

- On GitLab.com, a [top-level group Owner](../../../user/duo_agent_platform/model_selection.md#select-a-model-for-agentic-chat).
- On GitLab Self-Managed, an [instance administrator](../../../administration/gitlab_duo/model_selection.md#select-a-model-for-agentic-chat).

This gives organizations control over which models users can select when using Agentic Chat.
