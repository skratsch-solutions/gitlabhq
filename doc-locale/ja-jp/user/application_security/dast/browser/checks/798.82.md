## 説明 {#description}

レスポンスボディには、New RelicユーザーAPI IDのパターンに一致するコンテンツが含まれています。この値を露出すると、攻撃者がこのトークンによって付与されたすべてのリソースへのアクセスを取得する可能性があります。

## 修復 {#remediation}

キー漏洩に関するセキュリティインシデントの対処に関する一般的なガイダンスについては、GitLabドキュメントの[認証情報のインターネットへの露出](../../../../../security/responding_to_security_incidents.md#credential-exposure-to-public-internet)を参照してください。詳細については、[New RelicのAPIキーのローテーションに関するドキュメント](https://docs.newrelic.com/docs/apis/intro-apis/new-relic-api-keys/#rotate-keys)を参照してください。
