## 설명 {#description}

JSON 직렬화/주입 취약점을 확인합니다.

## 수정 {#remediation}

JSON 주입은 JSON 애플리케이션 또는 서비스의 논리를 조작하거나 손상시키는 데 사용되는 공격 기법입니다. JSON 메시지에 의도하지 않은 JSON 콘텐츠 및/또는 구조를 주입하면 애플리케이션의 의도된 논리를 변경할 수 있습니다. 또한 JSON 주입은 악성 콘텐츠를 결과 메시지/문서에 삽입할 수 있습니다.
