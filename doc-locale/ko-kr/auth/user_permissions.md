GitLab은 사용자 유형, 역할, 멤버십을 결합하는 포괄적인 권한 시스템을 사용하여 프로젝트와 그룹 내에서 수행할 수 있는 작업을 제어합니다. 사용자에게는 프로젝트와 그룹에서의 권한을 정의하는 역할이 할당됩니다. 멤버십과 관련 권한은 최상위 그룹에서 하위 그룹 및 해당 프로젝트로 계단식으로 적용됩니다.

사용자 유형은 GitLab 인스턴스 전체에서 다양한 액세스 수준을 가지고 있으며, 표준 권한을 가진 일반 사용자부터 전체 시스템 제어 권한을 가진 관리자까지입니다. 사용자는 조직의 요구 사항에 맞게 조정된 특정 권한이 있는 사용자 지정 역할을 가질 수도 있습니다.
