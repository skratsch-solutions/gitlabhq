이전에는 리포지토리 커밋 목록의 필터링이 제한되어 있어 긴 기록에서 특정 커밋을 찾기 어려웠습니다.

재설계된 커밋 목록에는 다음이 포함됩니다:

- 작성자, 커밋 메시지 또는 날짜 범위로 커밋을 필터링하고 검색합니다.
- 브랜치, 태그 또는 커밋 SHA와 같은 Git 리비전으로 목록을 필터링합니다.
- 더 쉽게 검색할 수 있도록 날짜별로 그룹화된 커밋입니다.
- 대규모 리포지토리에 대한 향상된 성능 및 페이지 매김입니다.
