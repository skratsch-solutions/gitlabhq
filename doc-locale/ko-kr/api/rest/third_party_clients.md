GitLab과 타사 API 클라이언트 라이브러리를 통합할 수 있습니다. 다음 라이브러리는 커뮤니티 멤버가 유지보수하며 GitLab에서 공식적으로 지원하지 않습니다. 버그 및 기능 제안은 각 프로젝트에 보고하세요.

이러한 통합에 대한 질문이 있으면 [GitLab 커뮤니티 포럼](https://forum.gitlab.com/)을 사용하세요.

관리자는 [로그 분석](../../administration/logs/log_parsing.md#print-top-api-user-agents)을 통해 이러한 API 클라이언트의 사용 현황을 모니터링할 수 있습니다.
