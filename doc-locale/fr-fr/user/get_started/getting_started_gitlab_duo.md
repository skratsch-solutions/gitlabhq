GitLab Duo est un assistant natif d'IA qui vous accompagne tout au long de votre workflow de planification, de développement et de sécurité. GitLab Duo comprend des fonctionnalités telles que Code Suggestions et Code Explanation, qui vous aident à écrire, réviser et modifier du code.

## Étape 1 :  Vérifier que vous avez accès à GitLab Duo {#step-1-ensure-you-have-access-to-gitlab-duo}

GitLab Duo nécessite une configuration par votre administrateur, votre groupe ou le propriétaire du projet.

Si vous rencontrez des problèmes pour accéder aux fonctionnalités de GitLab Duo, vos administrateurs peuvent vérifier l'état de l'installation.

Pour plus d'informations, consultez :

- [Activer GitLab Duo](../gitlab_duo/turn_on_off.md).
- [Détails du bilan de santé](../../administration/gitlab_duo/configure/_index.md#run-a-health-check-for-gitlab-duo).

## Étape 2 :  Essayer GitLab Duo Chat dans l'interface utilisateur {#step-2-try-gitlab-duo-chat-in-the-ui}

Pour commencer, essayez d'utiliser Chat dans l'interface utilisateur GitLab.

Accédez à un projet et, dans le coin supérieur droit, sélectionnez le bouton **GitLab Duo Chat**. Si ce bouton est disponible, cela signifie que tout est correctement configuré. Essayez de poser à Chat une question sur un ticket ou un merge request spécifique, ou sur GitLab en général.

Pour plus d'informations, consultez :

- [GitLab Duo Non-Agentic Chat](../gitlab_duo_chat/_index.md).

## Étape 3 :  Essayer d'autres fonctionnalités de GitLab Duo {#step-3-try-other-gitlab-duo-features}

GitLab Duo est disponible tout au long de votre workflow. De la planification des sprints au dépannage des pipelines CI/CD, de la rédaction de cas de test à la résolution des menaces de sécurité, GitLab Duo peut vous aider de diverses façons.

Les fonctionnalités auxquelles vous avez accès peuvent différer selon votre abonnement.

Pour plus d'informations, consultez :

- [Un arbre de décision pour vous aider à choisir les fonctionnalités GitLab Duo adaptées à votre workflow](../gitlab_duo/_index.md).
- [La liste complète des fonctionnalités GitLab Duo](../gitlab_duo/feature_summary.md).
- [Comment activer les fonctionnalités GitLab Duo encore en cours de développement](../gitlab_duo/turn_on_off.md#turn-on-beta-and-experimental-features).

## Étape 4 :  Se préparer à utiliser GitLab Duo dans votre IDE {#step-4-prepare-to-use-gitlab-duo-in-your-ide}

Essayez maintenant les fonctionnalités de GitLab Duo dans votre IDE. Dans VS Code et d'autres éditeurs, vous pouvez utiliser des fonctionnalités telles que GitLab Duo Chat, le flow de développement logiciel et Code Suggestions.

Pour commencer, installez une extension et authentifiez-vous auprès de GitLab.

Pour plus d'informations, consultez :

- [Configurer l'extension pour VS Code](../../editor_extensions/visual_studio_code/setup.md).
- [Configurer l'extension pour JetBrains](../../editor_extensions/jetbrains_ide/setup.md).
- [Configurer l'extension pour Visual Studio](../../editor_extensions/visual_studio/setup.md).
- [Configurer l'extension pour Neovim](../../editor_extensions/neovim/setup.md).
- [Utiliser le Web IDE](../project/web_ide/_index.md).

## Étape 5 :  Commencer à utiliser les fonctionnalités de l'IDE {#step-5-start-using-ide-features}

Enfin, testez GitLab Duo dans votre IDE.

- Code Suggestions recommande du code au fur et à mesure que vous tapez.
- Chat est disponible pour répondre à vos questions sur votre code ou tout autre sujet.
- Le flow de développement logiciel effectue des tâches en votre nom.

Vous pouvez choisir les langages de développement pour lesquels vous souhaitez des suggestions.

Pour plus d'informations, consultez :

- [Extensions et langages pris en charge](../project/repository/code_suggestions/supported_extensions.md).
