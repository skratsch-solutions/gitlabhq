GitLab fournit des feature flags pour activer ou désactiver des fonctionnalités spécifiques. Cette page contient une liste de tous les feature flags fournis par GitLab. Dans GitLab Self-Managed, les administrateurs GitLab peuvent [modifier l'état de ces feature flags](_index.md).

Pour obtenir de l'aide sur le développement de feature flags personnalisés, consultez [Créer un feature flag](../../operations/feature_flags.md#create-a-feature-flag).
